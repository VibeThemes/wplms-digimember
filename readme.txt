== Description ==

Integrate WPLMS Learning Management System with Digimember

WPLMS : https://wplms.io
DigiMember: https://digimember.de/

= More Information =

Visit the <a href="https://wplms.io">Wplms</a> for documentation, support, and information on getting involved in the project.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'WPLMS Digimember'
3. Activate WPLMS DigiMember from your Plugins page. 

= From WordPress.org =

1. Download WPLMS DigiMember.
2. Upload the 'bwplms-digimember' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate WPLMS DigiMember from your Plugins page.

= Once Activated =

1. Visit 'Settings > WPLMS DigiMember ' and adjust the settings. 
2. Create Memberships in Digimember plugin
3. Connect the memberships with the Courses
